This project is a **Python-based web scraper** that automatically extracts data from websites.
It uses `requests` to fetch HTML pages and `BeautifulSoup` to parse and extract useful information.

The scraper can be adapted to collect data such as article titles, product names, and prices.

Features
- Fetches HTML content from web pages
- Extracts structured data using BeautifulSoup
- Prints results to the console
- Optional export to CSV for further analysis

---

Technologies used
- Python 3
- Requests
- BeautifulSoup4
